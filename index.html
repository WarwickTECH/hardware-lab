The API

This is a list of the various end points available with the Hardware Lab API. Click the request to expand for more detail and examples.

GET /hardware
  • Returns a list of all hardware.
GET /hardware?status=
  • Returns a list of all hardware currently available/unavailable (true || false).
POST /hardware/random
  • Returns a random hardware currently available in the Harware Lab.
POST /hardware/:id
  • Request to borrow specifc hardware by id. Requires student id and email in payload.
GET /hardware/:id
  • Returns info about a specific hardware by its id.
GET /hardware/:name
  • Returns a list of all hardware of a specifc name, e.g. Arduino.
